# Synora-ai-chatbot

A lightweight AI chatbot that delivers intelligent conversational responses using a locally hosted language model with a clean web interface.

---

## 🚀 Features

- 💬 ChatGPT-style chatbot UI
- ⚡ Fast local responses using Ollama
- 🧠 Powered by Gemma 3 model
- 🔒 No API keys required
- 🌐 Simple Flask web interface
- 🖥 Fully offline AI chatbot
- 🧠 Maintains chat-style interaction

---

## 🛠 Tech Stack

- Python 🐍
- Flask 🌐
- HTML, CSS, JavaScript
- Ollama (Gemma 3)
- Frontend: HTML, CSS, JavaScript
